# capstone-project
A lightweight C++ console application that lets users easily explore directories, create, delete, copy, move, and search files, and manage file permissions , all through simple terminal commands on Linux and Windows systems.

Features

*List files and directories
Navigate between folders (cd, back)
Create and delete files or folders
Search for files recursively
Copy, move, or rename files
View and change file permissions (Linux & Windows supported)
Cross-platform support
